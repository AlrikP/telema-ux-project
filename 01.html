Bubbles

Activate: $('[data-bubble]').bubble();
data-bubble (required)
Required attribute to define bubble element
data-bubble-vpos
Defines bubble vertical position
Values: bottom, top (default: bottom)
data-bubble-hpos
Defines bubble horizontal position
Values: left, right (default: left)
data-bubble-disable-sideclick
Defines if bubble stays visible on document click
Values: true, false (default: false)
data-bubble-force-position
Defines if bubble sets its position to fit in area or viewport
Values: true, false (default: false)
data-bubble-area
Defines area element. If not set then viewport is used.
Bubble automatically sets its position not to appear outside the content area
data-bubble-toggler (required)
Required attribute to define bubble toggler element
data-bubble-content (required)
Required attribute to define bubble content element
data-bubble-close
Clicking on bubble doesn't close it
If target element has this attribute then bubble is closed

Custom selects, dropdowns and autocompletes

Dropdowns
Activate: $('[data-dropdown]').dropdown();
data-dropdown (required)
Required attribute to define dropdown element
data-dropdown-disable-sideclick
Defines if dropdown stays visible on document click
Values: true, false (default: false)
data-dropdown-disable-select-close
Defines if dropdown stays visible on option click
Values: true, false (default: false)
data-dropdown-toggler (required)
Required attribute to define dropdown toggler element
data-dropdown-content (required)
Required attribute to define dropdown content element
data-dropdown-indicator
Defines indicator element. Indicator changes its value on option element click.
Mainly used for custom selects
data-dropdown-option
Defines dropdown option element. Mainly used for custom selects.
data-dropdown-value
Defines dropdown option element value. This value is set to hidden input element.
data-dropdown-close
Clicking on dropdown doesn't close dropdown (except if element is option element)
If target element has this attribute then dropdown is closed
data-dropdown-hidden-input
Defines hidden input element. This is useful if custom select is used inside form
Option click triggers change:dropdown event on data-dropdown element
If some action is needed on select change then you can use something like this:
$('[data-dropdown]').on('change:dropdown', function() { console.log(attributes) });
